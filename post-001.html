here's everything I built before I had a brand

I've been building for a year without a website, a brand, or a single post about any of it. Three projects deep and nobody knew they existed. Today that changes.

What I built

Jarvis is a personal AI operating system running on GCP, Cloud Run, and PostgreSQL. It scrapes my Google Classroom assignments, detects new ones via iPhone OCR, generates drafts with an LLM, and schedules my work based on energy levels throughout the day. The scheduler fires every minute. The database has 9 tables. It's not a demo — it's real infrastructure.

Jarvis architecture diagram — add image

Gesture Control is hands-free Mac control using MediaPipe and OpenCV. Right hand swipe switches desktops. Fist pauses music. Eyes closed for 5 seconds puts the computer to sleep. Wave toggles brainrot mode for hands-free scrolling. Built with accessibility in mind — originally designed for people with mobility limitations.

Arduino SaaS is the simplest of the three. Describe what you want to build, get working Arduino code and a parts list with sourcing links. The demo project is Clear Safe — a keypad-locked safe with timer and PIN override that I built for my school's Innovation Nexus club.

Why now

I got into TKS this summer. I have 4 hackathons lined up. I'm starting a web design business for local Pleasanton businesses. And I realized I was building in a vacuum — shipping real things with nobody watching.

So I built the brand, launched the site, and this is post one.

What's next

Jarvis to sellable MVP. Gesture control research paper. Web design clients. Weekly devlogs documenting all of it. The chaos is the content.
