Overview

What it does

Most people who want to build Arduino projects get blocked by two things: writing the code and figuring out what parts to buy. This solves both.

You describe your project in plain language. The platform generates working Arduino code, builds a parts list with exact components, and links you to where to buy them from Amazon or Digikey.

The demo project is Clear Safe — an Arduino-powered safe with a 4x4 keypad, servo motor, timer lock, and PIN override. Built it in Innovation Nexus club at Quarry Lane.

Core features

  • Natural language project description input
  • Working Arduino code generation
  • Automatic parts list with quantities
  • Amazon and Digikey sourcing links
  • Clear Safe as viral demo project
  • Free tier: 3 generations per month
  • Paid tier: unlimited at $8/mo
